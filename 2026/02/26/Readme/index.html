关于这个网站的一些想法

千人千面，各有亮点。

2018年，习惯用不知道哪里摘抄来的歌词、小说语句表达感情；

2019年，沉浸在幻想与现实的边界，对她的喜欢或仰慕于当时的我，好坏参半；

2020年，情绪显形，摆脱稚气；

2021年，分别，新生；

2022年，emmm…, 某天晚上突然想好好学习（对22年的唯一印象）；

2023年，对于迈入18岁没有实感；

2024年，在无数人的支持下，面对一场命定的战役；

2025年，人生、感情的又一个转折口，一些从未有过的感觉被她人激发；

如今2026年，却又迈入一个全新的迷茫期……

初中的时候上数学课，思绪总是飘的很远。想象自己是小说里面自带主角光环的女主角，家财万贯，天赋惊人，作者用来作为全书冲突的感情波折在脑海里成了值得反复咀嚼的高潮。现在的我与那时的我并没有什么不同，同样不需要担心衣食住行，只顾在规律的人生路径里闷头。要说最不一样的，是少年心气多少被消磨，找不回几年前成绩带来的正反馈。

所以，诞生了这个仓库。

在此存储一些短期思考、览书观影的想法，希望能为未来的我留下一个青春的注脚。
